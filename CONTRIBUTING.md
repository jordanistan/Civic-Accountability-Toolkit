# Contributing

## Before you add anything
- Read `docs/sourcing-standard.md`
- Read `docs/safe-civic-use.md`
- Add a source note first

## Review checklist
- Is the claim supported?
- Is the source tier labeled honestly?
- Does the incident separate documented facts from unresolved allegations?
- Are linked people included only where the source actually ties them to the incident?

## Style
- Prefer short note titles
- One incident per note when possible
- Use plain English
- Avoid loaded language
