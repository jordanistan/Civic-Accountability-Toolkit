# Daily Workflow

## For ordinary users
1. Pick one incident from the incident index.
2. Check linked evidence and source tier.
3. Confirm what is established and what remains alleged.
4. Choose the right recipient: House, Senate, DOJ Public Integrity, or Inspector General.
5. Send a short, respectful, fact-based message.
6. Record what you sent and any reply.

## For contributors
1. Add or improve one source-backed note.
2. Link it to at least one incident and one issue.
3. Mark missing evidence clearly.
4. Run the sync script and validate links.
