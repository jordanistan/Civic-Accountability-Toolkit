# Contribution Guide

## Before you contribute
- Read the sourcing standard.
- Keep findings separate from allegations.
- Prefer official records, court orders, and document-backed reporting.

## Good contributions
- Adding a source note with clear factual extraction
- Improving an incident timeline
- Linking a person note to reliable evidence
- Clarifying what is still unproved

## Avoid
- Unsupported accusations
- Personal attacks
- Doxxing or harassment
- Copying entire news articles
