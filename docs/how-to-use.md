# How to Use

## For ordinary citizens
1. Start in `00_HOME.md`.
2. Pick one incident.
3. Read the linked evidence note.
4. Use the citizen letter template.
5. Send a respectful, fact-based oversight request.

## For contributors
1. Add a source note first.
2. Create or update an incident note.
3. Link the person, issue, and statute notes.
4. Mark what is documented vs unresolved.

## For maintainers
- Keep titles short and consistent
- Prefer one incident per note
- Use source tiers honestly
- Remove or quarantine weak sourcing
